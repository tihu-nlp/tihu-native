tihu mobile app mono repo

## Pre-requisites

- NodeJS
- Yarn

## Usage

Install dependencies

```sh
yarn
```

Run the app using expo

```sh
yarn start
```

Run the app and mock server in parallel

```sh
yarn start:mock
```

Run mock server stand-alone

```sh
yarn mock-server
```
