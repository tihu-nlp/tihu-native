# tihu-native-grpc

- NodeJS client code for tihu api
- Mock implementation of the tihu `gRPC` api for testing purposes

## Usage

```sh
yarn
yarn start
# with auto-restart
yarn dev
```

## Test

```sh
# launch the server
yarn start
# run tests
yarn test
# or run tests in watch mode
yarn test:watch
```
